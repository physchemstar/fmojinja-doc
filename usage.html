Usage¶

インプットファイルの生成¶

モジュール実行¶

python -m fmojinja.{program} [subcommand]でインプットファイルの生成が可能

以下はabinitmpに対してフラグメント分割のみを行うajfファイルの作成例

python -m fmojinja.abinitmp auto --pdb test.pdb > test.ajf

サブコマンドの種類やヘルプは以下で確認可能

# サブコマンドのリストの表示
python -m fmojinja.{program} -h

# サブコマンドのヘルプの表示
python -m fmojinja.{program} [subcommand] -h

出力ファイルの読み込み¶

モジュール実行¶

python -m fmojinnja.{program}.readerでアウトプットファイルの読み込みが可能

python -m fmojinja.abinitmp.reader test.cpf > test.csv

モジュールの読み込み¶

pandasに対するモンキーパッチを実装している。 pandas.read_xxx("output_file")で読み込むことができる。

import pandas as pd
import fmojinja.{program}.reader

pd.read_xxx("test.cpf")
